Matric Maths Mistakes Guide

A concise guide to avoid frequent Grade 12 maths mistakes and improve exam consistency.

1) Misreading the question

Underline command words and rewrite the ask before calculating.

2) Skipping algebra steps

Show transformations line by line so method marks are still earned when arithmetic slips happen.

3) No time strategy

Allocate time per mark and return to hard questions later.

4) Weak error review

Keep an error log by topic and trigger so revision follows patterns instead of guesswork.
